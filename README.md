# Interactive Navigation Menu

## SkillCraft Technology Internship - Task 01

### Project Overview

This project is an Interactive Navigation Menu developed using HTML, CSS, and JavaScript. The navigation menu remains fixed at the top of the page and provides an enhanced user experience through dynamic styling effects.

The menu changes its appearance when the user scrolls through the webpage and provides interactive hover effects on menu items. The website also includes smooth scrolling navigation and a responsive design suitable for different screen sizes.

---

## Features

✅ Fixed Navigation Bar

- Navigation menu remains visible at the top of the page while scrolling.

✅ Hover Effects

- Menu items change color and style when hovered over.

✅ Scroll Effects

- Navigation bar changes its background color and appearance when the page is scrolled.

✅ Active Menu Highlighting

- The current section is automatically highlighted in the navigation menu.

✅ Smooth Scrolling

- Smooth transitions between different sections of the webpage.

✅ Responsive Design

- Works properly on desktops, tablets, and mobile devices.

## How to Run the Project

1. Download or clone the repository.

```bash
git clone https://github.com/yourusername/SkillCraft_Task01.git
```

2. Open the project folder in Visual Studio Code.

3. Install the Live Server extension.

4. Right-click on `index.html`.

5. Select **Open with Live Server**.

6. The project will open in your browser.
